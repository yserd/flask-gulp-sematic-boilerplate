Welcome to flask-gulp-semantic-boilerplate

This is a starter/boilerplate Flask project.

It includes Semantic UI, Flask-SQLAlchemy, WTForms, Gulp, BrowserSync and various testing utilities out of the box.

Gulp is used to handle the frontend workflow. Included tasks are javascript minification and concatination, LESS compilation and image minification.

BrowserSync automatically reloads web pages when source code changes are saved.
